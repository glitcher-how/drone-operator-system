# Drone Operator System

Учебный проект по курсу **Методология кибериммунитета**.\
Система предназначена для автоматизации работы эксплуатанта беспилотных
летательных аппаратов (БВС).

Проект реализует базовые сценарии управления дронами: - регистрация
дронов - управление дронопортами - создание заказов-нарядов -
автоматический выбор дрона для выполнения миссии

------------------------------------------------------------------------

## Основные возможности

### Регистрация дронов

Оператор может добавить новый дрон в систему с параметрами: - тип
дрона - серийный номер - грузоподъёмность - дальность - заряд батареи -
срок действия сертификата

### Управление дронопортами

Система поддерживает регистрацию дронопортов: - название - локация -
вместимость - статус активности

Дроны могут быть привязаны к дронопортам.

### Создание заказов-нарядов

Оператор может создать заказ для выполнения миссии: - заказчик - тип
миссии - вес груза - точка отправления - точка назначения - время
выполнения

### Автоматический выбор дрона

Система автоматически выбирает подходящий дрон на основе: - статуса
дрона - уровня заряда - грузоподъёмности - наличия действующего
сертификата - привязки к дронопорту - близости к точке отправления

### Монитор безопасности

В системе реализован модуль Security Monitor, который выполняет
проверки: - сертификат дрона - уровень батареи - грузоподъёмность -
статус дрона - наличие активного дронопорта

Если дрон не проходит проверки безопасности, он не может быть назначен
на миссию.

------------------------------------------------------------------------

## Жизненный цикл миссии

Заказ: new → assigned → done

Состояние дрона: ready → busy → ready

## Установка и запуск

Установить зависимости:

    pip install -r requirements.txt

Запустить приложение:

    python app.py

После запуска система будет доступна по адресу:

    http://127.0.0.1:5000

------------------------------------------------------------------------

## Демонстрационный сценарий

1.  Создать дронопорт
2.  Зарегистрировать дрон
3.  Привязать дрон к дронопорту
4.  Создать заказ-наряд
5.  Нажать "Подобрать дрон"
6.  Завершить миссию
